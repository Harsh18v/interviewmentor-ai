# InterviewMentor AI

InterviewMentor AI is an AI-powered interview preparation platform that helps users prepare smarter for job interviews. Users can upload their resume, provide a job description, and add a self-description to generate a personalized interview preparation report.

The platform analyzes the provided information and generates insights, interview guidance, skill gap analysis, and preparation recommendations to help users improve their interview performance.

---

## Features

- User Authentication (Register/Login)
- Secure JWT-based Authentication
- Upload Resume
- Add Job Description
- Add Self Description
- AI-Generated Interview Preparation Report
- Match Score Analysis
- Personalized Interview Guidance
- Recent Reports Dashboard
- Responsive UI

---

### Backend

- Mongoose
- JWT Authentication
- bcrypt.js

---

## How It Works

1. User registers or logs into the platform
2. User uploads their resume
3. User enters the job description
4. User adds a self-description
5. AI analyzes the inputs
6. A personalized interview preparation report is generated
